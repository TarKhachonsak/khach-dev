# Daily Note — 26 มิถุนายน 2026

## งานวันนี้ (จาก Git Commits)

### 1. แก้ field สถานะการจัดส่งไม่แสดง — `BookSentFormComponent`

**ปัญหา:** field `_POST_STATUS` ไม่แสดงค่าใน `dx-form`

**Root Cause (ชั้น 1):** `ngOnInit()` เรียก `this.onSetDefaultPostStatus` โดยไม่มี `()` — function ไม่ถูก invoke

**Root Cause (ชั้น 2):** แม้แก้ `()` แล้ว ค่า set ลง `formData` โดยตรงแต่ `dx-form` ไม่รู้ว่าค่าเปลี่ยน → ไม่ re-render

**Root Cause (ชั้น 3):** `@ViewChild` ยังไม่พร้อมใน `ngOnInit` — ต้องใช้ `ngAfterViewInit`

**Root Cause (ชั้น 4):** parent ส่ง `formData` object ใหม่มาทีหลัง (หลัง API) → ค่าที่ set ไปหายไปเพราะ object เปลี่ยน

**Fix:** ใช้ `ngOnChanges` + `ngAfterViewInit` + `formRef.instance.updateData()`

### 2. เพิ่ม field คำนำหน้าชื่ออื่นๆ — `AccuserFormAddComponent` + `PersonCrimeFormAddComponent`

- เพิ่ม `dxi-item` สำหรับ field `PREFIX_OTHER` พร้อม `[visible]="formData.PREFIX_ID == 535"`
- เพิ่ม `onValueChanged` handler: ถ้าเลือก ID ที่ไม่ใช่ 535 → clear `PREFIX_OTHER`
- ใช้ arrow function เพื่อ preserve `this`
