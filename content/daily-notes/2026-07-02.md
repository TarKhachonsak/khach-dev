## ปัญหาที่เจอและวิธีแก้

- **DevExtreme dx-form custom component ไม่ render** → ต้องครอบด้วย `ng-template dxTemplate="item"` เสมอ
- **Duplicate `cellTemplate` name ใน dx-form** → DevExtreme template name ต้อง unique ทั่วทั้ง form ไม่งั้น template หลังๆ จะไม่ทำงานตามที่คาด
- **Node heap out of memory ตอน `ng build`** → เพิ่ม `--max-old-space-size` ให้ node โดยตรง (ผ่าน `node .../ng build` แทนเรียก `ng` ตรงๆ)
- **State ที่ประกาศไว้แต่ไม่เคย assign ค่า (`_taskFormCode`, `approveValue`)** → ทำให้ feature ใหม่ที่ไปอ้างอิง state นั้นพังเงียบๆ (ไม่ error แค่ logic ไม่ทำงาน) ต้องไล่ตาม chain ของ state ทั้งหมดก่อนเพิ่ม logic ใหม่ที่พึ่งพามัน
