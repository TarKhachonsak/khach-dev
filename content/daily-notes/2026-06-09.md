# 2026-06-09

## Programming Journal

วันนี้เจอ root cause สำคัญของ DevExtreme `dx-form`: Angular template ที่วางใน `dxi-item` ไม่ได้ render เสมอ เพราะ DevExtreme มี template pipeline ของตัวเอง

## สิ่งที่เกิดขึ้น

- `*ngIf` บน `dxi-item` ซ่อน content แต่ยังเหลือ card wrapper/border
- custom component ใน `dxi-item itemType="group"` ไม่ render เพราะไม่มี `ng-template dxTemplate="item"`

## Root Cause

- DevExtreme สร้าง wrapper/container ของ form item เอง
- `dxi-item` ต้องการ template slot ชื่อ `item` สำหรับ custom Angular content

## วิธีแก้

- ใช้ `ng-container *ngIf` ครอบ `dxi-item`
- ห่อ component ด้วย `<ng-template dxTemplate="item">`
