## งานที่ทำ

### 1. แก้ `onLoadNewAllegation` ไม่ถูกเรียก

**ปัญหา:** `this.onLoadNewAllegation` ใน `setTimeout` ขาด `()` ทำให้ฟังก์ชันไม่ถูก execute เลย

**วิธีแก้:**
- ลบออกจาก `setTimeout`
- ย้ายไปเรียกใน `LoadDataCitizenSQL` subscribe callback หลัง `_formData_ROD_FIR_TXN_ALLEGATION_GROUP` ถูก set แล้ว
- เปลี่ยน signature เป็น `cell?: any` รองรับทั้ง call จาก init (ไม่มี arg) และ grid row click

→ [[Function Reference Without Parentheses]], [[Fix Function Called in Wrong Lifecycle Phase]]

### 2. ลบ `addTempFields` ออก

**ปัญหา:** `addTempFields` สร้าง `_TEMP` suffix ให้ทุก field แล้ว replace `_formData_ROD_FIR_TXN_ALLEGATION_GROUP` ทั้งหมด ทำให้ field อื่น เช่น `PENALTY_INFO_ID` หายไปจาก form binding

**วิธีแก้:** ลบ `addTempFields` ออก แล้ว set เฉพาะ `OFFENSE_INFO_FULL_NAME_TEMP` (field เดียวที่ใช้จริงใน HTML)

→ [[Over-Mapping All Fields to TEMP Suffix]]

### 3. ซ่อน option A ใน radio เมื่อเคยเลือกไปแล้ว

**ปัญหา:** `flow_04_task_034_approve` radio มี option A (บำเพ็ญประโยชน์) ที่ต้องเลือกได้แค่รอบเดียว

**วิธีแก้:** เพิ่ม `disableFormioRadioOption()` ที่ทำ recursive tree walk บน formio JSON schema และ filter option ออก ก่อน bind ลง `[form]="formio"`

**Trigger:** ตรวจ `formioSubmission?.data?.flow_04_task_034_approve === 'A'` หลัง parse JSON

→ [[Patch Formio Schema Dynamically]]

## การตัดสินใจสำคัญ

- ใช้ patch JSON แทน customConditional เพราะ formio ไม่ support disable เฉพาะ option ใน radio โดยตรง
