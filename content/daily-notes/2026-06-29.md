## ปัญหาที่เจอและวิธีแก้

### 1. Leaflet Map — zoom out ทุกครั้งที่คลิกเลือกพิกัด

**สาเหตุ:** `positionChange.emit()` → parent อัปเดต `formData.lat/lng` → Angular detect change → `ngOnChanges` เรียก `this.map.setView([lat, lng], 13)` → zoom reset

**แก้ไข:** ลบ `this.map.setView(...)` ออกจาก `ngOnChanges` เหลือแค่ `this.marker.setLatLng(...)`

→ [[Leaflet Map Zoom Reset on Position Change]]

---

### 2. crime-scene-location-redesign — lat/lng field ไม่แสดงหลังเลือกพิกัด

**สาเหตุ:** ไม่ได้ wire `(positionChange)` event จาก `leaflet-map-overview`

**แก้ไข:**
- เพิ่ม `(positionChange)="onPositionChange($event)"` ใน template
- เพิ่ม method mutate `formData.lat/lng` โดยตรง
- ใช้ `_isMapVisible` flag ซ่อน field จนกว่าจะกดปุ่ม "ระบุพิกัด"

---

### 3. crime-scene-investigator-form — OFFICER_ID บันทึกเป็น 0

**สาเหตุ:** `setDefaultRole()` ใช้ spread → ตัด reference:
```typescript
this.formData = { ...this.formData, OFFICER_ROLE_ID: roleId }; // ❌
```

**แก้ไข:** mutate ตรงๆ:
```typescript
this.formData.OFFICER_ROLE_ID = roleId; // ✅
```

→ [[Angular Input Object Reference]]

### 5. Angular Template Spread Operator Error

**แก้ไข:** ย้าย logic ไปเป็น getter ใน TS

→ [[Angular Template Does Not Support Spread Operator]]
